What works today: a full C preprocessor pipeline (line splicing, comment removal, `#include "file.h"` and `#include <file.h>`, object-like and function-like `#define` with parameter substitution and recursive expansion, variadic macros with `__VA_ARGS__`, macro stringification `#param`, token pasting `##`, `#undef`, conditional compilation `#ifdef`/`#ifndef`/`#if`/`#elif`/`#else`/`#endif` with full recursive-descent expression evaluation including `defined()`, macro expansion, arithmetic, bitwise, shift, logical, and comparison operators, `#error`, predefined macros `__FILE__`/`__LINE__`/`__STDC__`/`__STDC_VERSION__`/`__CLAUDEC99__`, command-line `-D` and `-I` flags); statements (if/else, while/do-while/for, switch/case/default, break/continue/goto, blocks with shadowing, expression statements); the full signed integer hierarchy (char/short/int/long) plus all four unsigned variants (unsigned char/short/int/long) with unsigned-aware arithmetic, division, right shift, and comparisons; the U/u integer literal suffix; void type and void* generic pointer (implicit conversion to/from any non-function pointer type, void functions, bare return, int f(void) zero-parameter form, dereferencing/arithmetic/subscripting void* rejected); minimal const qualifier (base-type scalars, direct assignment to const lvalues rejected); integer promotions, Usual Arithmetic Conversions for mixed signed/unsigned operands, casts, and integer literal typing with L suffix; multi-function programs with forward declarations, variadic function declarations and calls (at least the fixed parameter count required, xor eax,eax emitted per SysV AMD64 convention), and SysV-AMD64 calls (≤6 args) including boundary conversions; pointers (declarations, &/* as both rvalue and lvalue, assign-through-pointer, pointer parameters and return types, full pointer arithmetic including difference yielding long, NULL as null pointer constant); arrays (declarations, indexing, multi-level subscript a[i][j], decay-to-pointer, file-scope brace/string initializers with size inference, char-array initialization from string literals with explicit or inferred size); string literals (tokenization, static-data emission, decay to char*, decoded escapes); character literals (full simple-escape set, int typing in any integer context); all bitwise operators (& ^ | ~ << >>), remainder %, all eleven compound assignment operators; sizeof (type names, expressions, whole arrays/structs — operand not evaluated, sizeof(void) rejected); conditional (ternary) operator; comma operator; comma-separated init-declarator lists; function-declaration compatibility checks; file-scope global variable declarations (uninitialized in .bss, initialized in .data including struct aggregate initializers, RIP-relative addressing); storage-class specifiers extern and static; parenthesized declarators; abstract pointer parameters in prototypes (e.g. char*); function pointer declarations/assignment/indirect calls (fp(args) and (*fp)(args)); unified declarator with declaration specifier list parsing accepting const qualifier; typedef aliases for scalar/pointer/array/function-pointer and complete struct types with block-scope shadowing, unsigned scalar typedefs preserving signedness; enum declarations with compile-time constant folding; structs (definition, natural-alignment layout, sizeof, dot/arrow member access in rvalue and lvalue contexts, brace initializer lists with zero-fill, whole-struct assignment and copy-initialization, pointer-to-struct parameters with field mutation, nested structs, arrays of structs, typedef aliases, incomplete forward declarations for self-referential types).

What's still out of scope: floats, for-loop initializer declarations, variadic function *definitions* (va_list/va_start/va_arg/va_end/<stdarg.h>), octal/hex character escapes, multi-character and wide-character literals, >6-arg calls, unions, bit-fields, flexible array members, compound literals, pointer-level const enforcement (writes through const*, conversion const-correctness), typedef enum, enum constant expressions beyond integer/character literals, functions returning function pointers, #pragma, __DATE__/__TIME__ predefined macros, GNU variadic macro extensions (__VA_OPT__, comma deletion).
