● Here's the expression parser hierarchy, from lowest to highest precedence:                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                          
  parse_expression                                                                                                                                                                                                                                                                        
  │  comma operator:  a , b , c  →  (a , b) , c                                                                                                                                                                                                                                           
  │  left-associative                                                                                                                                                                                                                                                                     
  └─► parse_assignment_expression                                                                                                                                                                                                                                                         
         assignment:  x = e  |  x += e  |  x -= e  | ...                                                                                                                                                                                                                                  
         right-associative                                                                                                                                                                                                                                                                
      └─► parse_conditional                                                                                                                                                                                                                                                               
             ternary:  cond ? true : false                                                                                                                                                                                                                                                
             right-associative                                                                                                                                                                                                                                                            
          └─► parse_logical_or                                                                                                                                                                                                                                                            
                 ||                                                                                                                                                                                                                                                                       
                 left-associative                                                                                                                                                                                                                                                         
              └─► parse_logical_and                                                                                                                                                                                                                                                       
                     &&                                                                                                                                                                                                                                                                   
                     left-associative                                                                                                                                                                                                                                                     
                  └─► parse_bitwise_or                                                                                                                                                                                                                                                    
                         |                                                                                                                                                                                                                                                                
                         left-associative                                                                                                                                                                                                                                                 
                      └─► parse_bitwise_xor                                                                                                                                                                                                                                               
                             ^                                                                                                                                                                                                                                                            
                             left-associative                                                                                                                                                                                                                                             
                          └─► parse_bitwise_and
                                 &                                                                                                                                                                                                                                                        
                                 left-associative                                                                                                                                                                                                                                         
                              └─► parse_equality                                                                                                                                                                                                                                          
                                     ==  !=                                                                                                                                                                                                                                             
                                     left-associative                                                                                                                                                                                                                                     
                                  └─► parse_relational                                                                                                                                                                                                                                    
                                         <  <=  >  >=                                                                                                                                                                                                                                     
                                         left-associative                                                                                                                                                                                                                                 
                                      └─► parse_shift                                                                                                                                                                                                                                     
                                             <<  >>                                                                                                                                                                                                                                       
                                             left-associative                                                                                                                                                                                                                             
                                          └─► parse_additive                                                                                                                                                                                                                              
                                                 +  -                                                                                                                                                                                                                                     
                                                 left-associative                                                                                                                                                                                                                         
                                              └─► parse_term

                                                             + - ! ~ ++ -- * &

                                                             right-associative                                                                                                                                                                                                            
                                                          └─► parse_postfix                                                                                                                                                                                                               
                                                                 expr++  expr--  expr[i]                                                                                                                                                                                                
                                                                 left-associative                                                                                                                                                                                                         
                                                              └─► parse_primary
                                                                     integer literal                                                                                                                                                                                                      
                                                                     char literal                                                                                                                                                                                                         
                                                                     string literal
                                                                     identifier                                                                                                                                                                                                           
                                                                     function call                                                                                                                                                                                                      
                                                                     ( expression )
                                                                                                                                                                                                                                                                                          
  Key design points:
  - Each level calls the level below it for its operands                                                                                                                                                                                                                                  
  - Recursion direction encodes associativity: right-recursive = right-associative, iterative loop = left-associative                                                                                                                                                                     
  - parse_expression and parse_assignment_expression are the only two levels that are right-recursive                
  - Function call arguments use parse_assignment_expression directly (skipping the comma level) so f(a, b) stays two arguments                                                                                                                                                            
  - Parenthesized expressions in parse_primary call back up to parse_expression (the top), which is what makes (a, b) a valid comma expression inside a call
