# Generated Artifacts

When generating artifacts for a stage, use these templates:

| Artifact  | Template |
| --- | --- |
| Stage Kickoff | ${CLAUDE_SKILL_DIR}/template/stage-tickoff-template.md |
| Transcript | ${CLAUDE_SKILL_DIR}/template/transcript-format.md |
| Milestone Summary | ${CLAUDE_SKILL_DIR}/templates/milestone-summary.md |

Rules:

- Read the appropriate template before creating the artifact.
- Follow the template exactly unless the stage spec says otherwise.
- Use `STAGE_LABEL` in file names.
- Use `STAGE_DISPLAY` in artifact titles.
